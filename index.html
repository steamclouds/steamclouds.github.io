✨ Core Features

🎯 Automatic Download & Install Manifest

Automatically download and install Steam manifests (Steam Unlocked). One-click install for games & DLC so your library fills instantly.

🧩 Simple & Minimalist Design

A lightweight, focused launcher with a clean UI — no clutter, just the tools you need.

➕ Super Easy to Add Games

Click Add Games once, restart Steam, and your library will be populated automatically.

🛡️ Super Safe & Free

Local-only tool with no hidden fees and no automatic data collection — free to use for everyone.

🔄 Up-to-Date Tools

Regular updates to keep compatibility with new Steam changes and improve stability.

📂 Import Lua & Manifest

Drag & drop Lua or manifest files (your own or shared by others) directly into the app to add games instantly.

📖 How to Use Steam Clouds

  1. Visit steamclouds.github.io
  2. Download the latest Steam Clouds version
  3. Run SteamClouds.exe
  4. Go to Install Steam Clouds
  5. Follow the installation steps until finished
  6. Restart your Steam client
  7. Have fun — enjoy your populated library 🎮

🛡️ Safety Statement

This software is a pure local green tool — it runs without administrator permission and works locally on your machine. We do not collect personal information because we do not operate servers (except for an optional Steam Clouds Discord bot that helps manage manifests on a dedicated server).

There is no user login. Some antivirus software may show false positives (Unknown Publisher). This is a common false positive for unsigned local apps — rest assured the tool itself is local and safe to use.
